# Changelog

## 1.0.0b0 (beta)

- Первый публичный beta-релиз macOS-версии `Zapret Hub`.
- Единый UI-хаб для управления `ByeDPI` / `SpoofDPI` и `TG WS Proxy`.
- Реализован системный прокси-пайплайн для локального трафика.
- Добавлен выбор движка обхода с runtime-переключением.
- Реализован режим работы через menu bar при закрытии окна с активными обходами.
- Добавлены живые логи компонентов и диагностика состояния.
- Настройки и пользовательские данные вынесены в `~/Library/Application Support/Zapret Hub Mac`.
- Добавлена проверка обновлений из GitHub:
  - при запуске приложения;
  - автоматически каждые 5 часов;
  - показ диалога с переходом на страницу релиза.
